# AI Workplace Productivity Assistant

## Project Overview

The AI Workplace Productivity Assistant is a web-based AI solution designed to help professionals automate repetitive workplace tasks and improve productivity. The platform provides tools for email generation, meeting note summarization, task planning, research assistance, and AI-powered workplace support.

Developed as part of the **CAPACITI AI Skill Accelerator Programme**, this project demonstrates practical AI implementation, prompt engineering, workplace automation, and responsible AI usage.

## Setup Instructions

### Access the Project

1. Open the deployed application link.
2. Navigate through the sidebar to access:

   * Dashboard
   * Email Generator
   * Meeting Notes Summarizer
   * Task Planner
   * Research Assistant
   * AI Chat Assistant

### Using the Application

#### Generate Emails

* Enter the email purpose.
* Select a tone.
* Click **Generate Email**.

#### Summarize Meeting Notes

* Paste meeting notes.
* Click **Summarize**.
* Review key points and action items.

#### Manage Tasks

* Create new tasks.
* Mark tasks as completed.
* Monitor progress through the dashboard.

#### Research Topics

* Enter a research query.
* Review AI-generated summaries and insights.

#### Chat with AI Assistant

* Type workplace-related questions.
* Receive AI-generated responses instantly.

## Responsible AI Statement
